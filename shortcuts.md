# Shortcuts

| Key | Action |
|-----|--------|
| M | Toggle recording |
| I | **AI Vision**: Capture screen & Solve (Code/MCQ) |
| C | Copy generated code to clipboard |
| Numpad * | Enable/Disable all shortcuts |
| Numpad - | Hide window and disable all shortcuts (Stealth) |
| Insert | Restore window (only works when hidden) |
